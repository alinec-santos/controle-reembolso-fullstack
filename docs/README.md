# Documentação do Projeto

Esta pasta contém a documentação do sistema de controle de solicitações de reembolso.

## Conteúdo

- **01-documentacao-backend**  
Este documento explica em detalhe tudo que existe dentro do projeto: as ferramentas utilizadas, a
estrutura de pastas (o que cada arquivo faz), o fluxo completo de uma requisição HTTP, a modelagem do
banco de dados, as regras de negócio, as validações e os testes automatizados. O objetivo é que você
consiga se virar dentro do projeto com autonomia.
